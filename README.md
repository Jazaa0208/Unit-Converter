# Unit Converter

A simple menu-driven Python application that performs Length, Weight, and Temperature conversions.

## Features

### Length Conversions

* Meter → Kilometer
* Meter → Centimeter
* Kilometer → Meter

### Weight Conversions

* Gram → Kilogram
* Kilogram → Gram

### Temperature Conversions

* Celsius → Fahrenheit
* Fahrenheit → Celsius

## Concepts Used

* Python Loops
* Nested Loops
* Conditional Statements
* User Input Handling
* Input Validation
* Mathematical Calculations

## How to Run

```bash
python unit_converter.py
```
